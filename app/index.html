How to

Simple usage

HTML | Include css to the header

<header>
  <link rel="stylesheet" href="your-css-path/easy-notification.css">
</header>
HTML | Include js to the end of the body 

  ...
  <script src="your-js-path/easyNotification.js"></script>
</body>
JS | Usage 

let notification = new EasyNotification();

notification.notify('Everything is good', 'success');
notification.notify('Something wend wrong', 'error');

Custom usage

JS | Paste custom config into configuration method before calling

let notification = new EasyNotification();
let config = {
    'autoHide': true,
    'showProgressBar': true,
    'autoHideDuration': 10000,
    'globalPosition': 'top-left',
    'showAnimation': 'fadeInBounce',
    'hideAnimation': 'fadeOutScaleDown'
};
notification.configuration(config)

notification.notify('Everything is good', 'success');
notification.notify('Something wend wrong', 'error');
